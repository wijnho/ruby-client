# Zumata

For interaction with the Zumata API 2.0
http://developer.zumata.com/doc/zumata-api-2-0-documentation

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'zumata'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zumata
