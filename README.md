# Zumata

For interaction with the Zumata API 2.0
http://developer.zumata.com/doc/zumata-api-2-0-documentation

## Configuration
```
Zumata.configure do |config|
  config.api_url = "http://api.zumata.com"
  config.api_key = "YOUR_SECRET_API_KEY"
end
```
